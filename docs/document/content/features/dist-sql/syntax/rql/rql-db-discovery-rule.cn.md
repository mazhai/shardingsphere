## 定义

```sql
SHOW DB_DISCOVERY RULES [FROM schemaName]              
```

## 说明

| 列              | 说明            |
| --------------- | -------------- |
| name            | 规则名称         |
| dataSourceNames | 数据源名称列表    |
| discoverType    | 数据库发现服务类型 |
| discoverProps   | 数据库发现服务参数 |
